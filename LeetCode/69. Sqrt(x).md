#####Problem
Implement int sqrt(int x).

Compute and return the square root of x.

x is guaranteed to be a non-negative integer

#####Solution
```python
class Solution(object):
    def mySqrt(self, x):
        """
        :type x: int
        :rtype: int
        """
        
        if x < 2:
            return x
        
        left, right = 1, x / 2
        while left < right - 1:
            mid = left + (right - left) / 2
            if mid > x / mid:
                right = mid
            else:
                left = mid
        
        
        return right if x / right == right else left
```
