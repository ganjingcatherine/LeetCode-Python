#####Problem
Implement pow(x, n).

<pre>
Example 1:

Input: 2.00000, 10
Output: 1024.00000
Example 2:

Input: 2.10000, 3
Output: 9.26100
</pre>

######Solution
```python
class Solution(object):
    def myPow(self, x, n):
        """
        :type x: float
        :type n: int
        :rtype: float
        """
        
        if n == 0:
            return 1
        elif n < 0:
            x = 1.0 / x
            n *= -1
        res = 1
        while n != 0:
            if n & 1:
               res *= x
            x *= x
            n = n >> 1
        return res
        
        
        
```
